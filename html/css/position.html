定位 position 属性学习

position 属性指定一个元素的定位类型，有以下几种属性值

值 描述
absolute 绝对定位，相对于 static 定位以外的第一个父元素进行定位。
fixed 绝对定位，相对于浏览器窗口
relative 相对定位，相对于其正常位置进行定位
static 默认值
inherit 规定应该从父元素继承 position 属性的值

static 默认定位

static 是默认定位，按照浏览器的默认值对元素进行排版

relative 相对定位

相对于其正常位置进行定位，如果不进行特殊配置和 static 并无区别

设置了relative 属性 并设置了top = -20 和 left = 20

fixed 绝对定位

一个固定定位（position属性的值为fixed）元素会相对于视窗来定位，这意味着即便页面滚动，它还是会停留在相同的位置。和 relative 一样， top 、 right 、 bottom 和 left 属性都可用。

绝对定位

可以看到右下角的一个浮动窗口，随着页面滑动一直定位在右下角,同样的还有顶部的标题栏，它也会随着窗口滑动固定在顶部

一个固定定位元素不会保留它原本在页面应有的空隙即脱离文档流。
