  • 空心圆点 Circle

  • 实心方块
    • list-style-type: lower-roman

也可以分别对条目指定样式：

  • none 无标记
  • disc 实心圆（默认）
  • circle 空心圆
  • square 实心方块
  • decimal 数字
  • decimal-leading-zero 0开头的数字标记
  • lower-roman 小写罗马数字
  • 。。。
